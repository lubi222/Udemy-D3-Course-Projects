For your third class project, you'll be building a line chart that looks like this: 

Once again, you should tackle this project in small pieces, gradually building your chart up from a static implementation. For this project, I’m starting you off with the line chart code that I showed you in this section. 

Your page has already been set up with everything that you’ll need.

  1. Change the source of your data from the example data in data/example.json  to our cryptocurrency data in data/coins.json . 
    1. Make a console log of the data that you're loading in, to see what the structure of your data looks like.
    2. You’ll need to clean the data so that we can use it with D3. Write some code that iterates over every coin, and converts our values to be used with D3.
  2. Start off by rendering a static line chart to the screen using the data for just one coin (e.g. using the “bitcoin” array in your data).
  3. Create an update()  function, which you can call whenever an event gets fired.
    1. Move any code into this update()  function that will need to adjust to a changing dataset.
    2. Append your line path once at the top of your file, then adjust its "d"  attribute every time the graph updates, using a different line()  function.
    3. Add transitions to your line and your axes, so that your visualization updates seamlessly.
  4. Add a listener to the change  event of the #coin-select  and #var-select  select boxes.
    1. Whenever the selections are changed, this should fire your update()  function.
    2. Make some edits to your update()  function, so that the coin and variable that you're using is coming from these select boxes.
    3. Change the text of your y-axis label depending on the variable that we're looking at.
  5. Add a date slider to the #date-slider  div.
    1. Initialize the slider with some sensible values, and attach a listener onto the slide  event to trigger your update() function.
    2. In your update()  function, use the filter()  method of your data array to only include dates within the range set by your slider.

You might find this code useful for your y-axis ticks, so that the numbers can be formatted with different abbreviations (e.g. 1.5K, 2.2M, 43B). You can use this with the tickFormat() method. 

var formatSi = d3.format(".2s");
function formatAbbreviation(x) {
    var s = formatSi(x);
    switch (s[s.length - 1]) {
        case "G": return s.slice(0, -1) + "B";
        case "k": return s.slice(0, -1) + "K";
    }
    return s;
}
